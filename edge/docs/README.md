## üí° About

**ChatGPT Widescreen Mode** adds widescreen + fullscreen modes to ChatGPT, enhancing your chat sessions by reducing scrolling & creating a more immersive viewing experience.

## üöÄ Installation

1. Install [ChatGPT Widescreen Mode](https://edge.chatgptwidescreen.com) Edge add-on

2. Visit [chat.openai.com](https://chat.openai.com) as normal & mode toggles are added to chatbar!

## üåê Compatibility

ChatGPT Widescreen Mode is also available as an [Chrome add-on](https://chrome.chatgptwidescreen.com) and a [Greasemonkey userscript](https://greasyfork.org/en/scripts/461473-chatgpt-widescreen-mode) (via [Tampermonkey](https://www.tampermonkey.net/).

- [Google Chrome](https://www.chrome.com)
